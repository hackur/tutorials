Webduino Package D ( 電子材料包 )

Webduino Package D 是比較進階應用的材料包，裏頭可以使用紅外線接收器紀錄遙控器訊號，進一步控制家電用品 ( 例如電視機 ) 或玩具 ( 例如機器人 )，而三軸加速度計可以偵測三維空間的加速度變化，十分廣泛的應用在各種體感遊戲或虛擬實境的應用，搭配搖桿的輔助，就可以做出許多形形色色的多媒體應用。( Package D 目前只支援 Webduino Fly 和 Arduino UNO，要注意不要買錯囉~ )
