Webduino Package A ( 電子材料包 )

Webduino Package A 是入門的零件材料包，對應的所有範例在官方網站均有提供，當你使用 Package A ，你可以從中學習到基本的電子電路，從點亮一顆 LED 燈到自由操控三顆 LED 燈，從七彩霓虹燈玩轉到按鈕開關，甚至可以用超音波傳感器控制 Youtube 的影音播放，以上都只是一小部分的創意應用，更多想像的空間就留待你的創造力去實現。
