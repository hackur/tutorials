Webduino 智慧插座 - 嚐鮮體驗套件

本套件由 Webduino Smart 和智慧插座套件組合而成！透過 Smart 開發板控制插座，讓插座也能隔空被控制！

Webduino 智慧插座 - 經濟實惠套件

本套件內含兩塊 Webduino 開發板（ 馬克一號、Smart ）與智慧插座套件，透過此套件不僅一次玩轉兩塊開發板，進而學習兩塊板子互動、溝通，真正落實雲端監控、智慧生活。

Webduino 智慧插座 - 實戰開發套件

本套件為實戰智慧插座最完整的套件！內含三塊 Webduino 開發板（ 馬克一號、FLY+UNO、Smart ）、智慧插座與擴充套件，搭配「實戰家電物聯網」一書，數十種插座神奇控制法都能實現。

Webduino 開發板

Webduino 馬克 1 號

經過千錘百鍊的電路設計，Webduino 馬克 1 號終於以一身紅的姿態出現，對於手邊已經有傳感器或電子零件的開發者，可以直接藉由馬克 1 號感受 Webduino 的方便和美好。

Webduino Fly ( UNO 雲端擴充板 )

當 Arduino UNO 套用 Webduino Fly 之後，立即就變成具備透過 Wifi 控制的能力，立即擁有完整的 Webduino 開發功能，只需用 Web 技術，就可以控制 Arduino 相容的感應模組。

Webduino Fly + Arduino UNO

Arduino UNO + Webduino Fly 是目前市面上最方便的物聯網解決方案，透過 Webduino Fly 的輔助，你可以自行燒錄對應的韌體，讓開發板具備獨一無二的功能，展現您與眾不同的創造力！

Webduino Smart ( 雲端開發板 )

Webduino Smart 是 2016 年推出的物聯網開發板，同時具備連上網際網路和 WebSocket 操控的能力。

Webduino 開發者套件

Webduino 開發者套件 - 基本包

Webduino 開發套件基本包是 Webduino 馬克 1 號與 Webduino Package A 互相搭配的開發套件，可以實作許多電子電路基礎範例，以及開發超音波傳感器的有趣應用。

Webduino 開發者套件 - 威力加強包

Webduino 開發套件威力加強包是 Webduino 馬克 1 號、 Webduino Package A、Webduino Package B 互相搭配的開發套件，可以開發各種基礎應用以及串接各式各樣的傳感器。

Webduino 開發者套件 - Dr. Smart

Dr. Smart 套件組內含 Webduino Smart 開發板與多個電子元件、傳感器，是 Webduino 系列產品中最精簡的一個組合。

Webduino 主題式套件

Webduino 智慧植栽套件

Webduino 智慧植栽套件是專門針對植物栽培的套件，，內含繼電器、水泵、光敏電阻與土壤濕度偵測...等植栽專用零件組合。

Webduino 智慧插座擴充套件

本套件為實現 Webduino 智慧插座各種創意實作的擴充套件組，搭配著「Webduino智慧插座」一同實作，亦可跟隨著「實戰家電物聯網」書中的教學步驟，一步步帶您從行動裝置控制到串接網頁資料庫等多項有趣的互動方式。

Webduino 電子材料包

Webduino 豪華套件包

Webduino 豪華套件包裏頭包含了 13 個電子零件模組以及 7 種傳感器，當您擁有了豪華套件包，就代表您已經具備了跨足物聯網開發的進階功力！

Webduino 基本套件 Plus

Webduino 基本套件 Plus 是原本 Webduino Package A 的加強版，額外增加了 LED 點矩陣、蜂鳴器與震動開關，可以做出更多有趣的應用。

Webduino 智慧插座套件 ( 材料包 )

Webduino 智慧插座是將我們一般生活裡常見的插座改裝，透過馬克一號或是 Fly 與繼電器的輔助，讓插座可透過無線網路連上雲端。

Webduino 公仔自走車 ( 材料包 )

Webduino 公仔自走車是 Webduino 自主研發的產品，產品內附贈四組紙公仔造型，可以讓多台車互相連動，或結合行動裝置做出各種應用。

Webduino 自走車循跡升級套件 ( 材料包 )

自走車循跡升級套件可以讓您的 Webduino 自走車具備沿著路徑移動的能力，同時也會紀錄軌跡並重新播放路徑。

Webduino 擴充套件 S

Webduino 擴充套件 S 是傳感器的零件組，包含了溫濕度、光敏電阻、土壤濕度偵測、人體紅外線偵測以及聲音偵測五種傳感器，透過外部資訊的接收，更能掌握環境的一切變化。

Webduino 擴充套件 M

Webduino 擴充套件 M 是偏向動力驅動的零件組，內含繼電器、雙馬達驅動板以及可變電阻，各種不同的玩具與家電改造，都可透過擴充套件 M 來實現！

Webduino 擴充套件 P

Webduino 擴充套件 P 是針對進階玩家所準備的擴充套件，由於腳位較多，因此僅支援 Webduino Fly 以及 Arduino UNO，許多精彩的創意，都可以透過擴充套件 P 來實現喔！
