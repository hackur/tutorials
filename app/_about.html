关于我们

我们是一个高雄的热血创意团队，成员有韧体工程师、网页前后端工程师以及视觉设计师，凭着一股对网页技术无止尽的爱，在 2015 年，实现了真正用 Web 串接 Arduino 的完整过程，纯粹撰写 HTML 与 Javascript 的网页语法，就能够玩转各式各样的传感器，为广大的开发者和网页设计师，创造了前所未有的开发模式，我们深信在不久的将来，网页技术将引领开发者的潮流，带来各式各样的创意与契机！

我们每个月都会举办活动，与各大开发者社团也都有相关交流，在平均下来一个月至少举办三场活动 ( 教师研习、社团、展览...等 )，于 2015 年更参与了台湾知名的大型活动 ( JSDC、WebConf、MOPCON、MakerFaire...等 )，我们相信参与这些大型的活动，可以从许多前辈、先进与同好身上，获得许多宝贵的改善建议，同时也能将自己的技术贡献给广大的开发者们。
